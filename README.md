# 📰 Gazete Projesi (Next.js)

Bu proje, Yusuf Cidik tarafından Next.js (App Router) kullanılarak geliştirilmiş bir haber portalı uygulamasıdır.

## 🚀 Kurulum ve Çalıştırma

Boyutu nedeniyle `node_modules` klasörü repoya dahil edilmemiştir. Projeyi çalıştırmak için şu adımları izleyin:

1. **Repoyu klonlayın:**
   git clone https://github.com/YusufCidik/gazete.git

2. **Proje klasörüne girin:**
   cd gazete

3. **Gerekli paketleri yükleyin:**
   npm install

4. **Projeyi başlatın:**
   npm run dev

## 🛠️ Detaylar
- **Frontend:** Next.js 15, Tailwind CSS
- **Font:** Geist (Vercel)
- **Local URL:** http://localhost:3000
